# 🏡 Xlent Builder – Luxury Real Estate Website

A modern and fully responsive luxury real estate website built using HTML, CSS, and JavaScript.  
Designed for premium property showcasing with elegant UI, smooth animations, dark/light mode, responsive navigation, property sections, gallery showcase, and contact integration.

---

## ✨ Features

- 🌙 Dark / Light Theme Toggle
- 📱 Fully Responsive Design
- 🏠 Premium Real Estate Landing Page
- 🖼️ Property Gallery Section
- 📍 About & Services Sections
- 📞 Direct Call CTA Button
- 🍔 Mobile Hamburger Navigation
- ⚡ Smooth UI Transitions & Animations
- 🎨 Luxury Modern Design
- 🚀 Fast & Lightweight

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts
- Responsive Flexbox Layout
